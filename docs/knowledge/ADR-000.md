# ADR-000 : Choix de la stack

**Statut** : Accepté  
**Date** : 2026-07-22  
**Dernière mise à jour** : 2026-07-22  
**Approbation** : Équipe

## Contexte
Le projet doit être rapide, moderne, facile à déployer et maintenable.

## Décision
- **Frontend** : Vite 8 + React 19 + TypeScript 6
- **CSS** : Tailwind CSS 3
- **Backend** : Supabase (Auth, BDD PostgreSQL, Storage)
- **Déploiement** : Netlify
- **Tests** : Vitest + Playwright

## Version note
Les versions ci-dessus reflètent les dépendances installées au moment de la rédaction. Après chaque `npm install` significatif, ce document doit être vérifié et mis à jour si nécessaire.

## Conséquences
- Build rapide (Vite/Rolldown)
- Typage strict (TypeScript)
- Authentification et BDD intégrées (Supabase)
- Déploiement continu simplifié (Netlify)
