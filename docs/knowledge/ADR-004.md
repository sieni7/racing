# ADR-004 : Design System

**Statut** : Accepté  
**Date** : 2026-07-22  
**Approbation** : Équipe

## Contexte
Le projet a besoin d'une identité visuelle cohérente et de composants réutilisables pour le site du Racing Club de Bingerville.

## Décisions

### Palette de couleurs
- **Primary** : `#F97316` (orange club) — boutons, liens, accents
- **CTA** : `#EA580C` (orange foncé) — hover, états actifs
- **Bleu club** : palette `blue` (500/600/700/800/900)
- **Mode sombre** : support via `darkMode: 'class'`, classes `dark:` partout

### Typographie
- **Titres (display)** : `Space Grotesk` (Google Fonts, importé dans `index.html`)
- **Corps (sans)** : `Inter` (Google Fonts)
- Défini dans `tailwind.config.js` via `fontFamily`

### Composants UI
- Tous les composants réutilisables dans `src/components/ui/`
- Atomiques : un composant = une carte (PlayerCard, MatchCard, NewsCard, StaffCard)
- Squelettes de chargement : `Skeleton.tsx` avec variantes `CardSkeleton`, `ListSkeleton`
- Images : `loading="lazy"` sur tous les `<img>`, fallback `onError` vers initiales/placeholder

### Layout
- Navbar sticky avec détection de scroll (> 50px)
- Footer avec liens club et réseaux sociaux
- Container max-width `max-w-7xl` ou `max-w-4xl` selon la page

## Conséquences
- Design cohérent et maintenable
- Pas de dépendance à une bibliothèque UI externe
- Le mode sombre double le nombre de classes tailwind mais améliore l'accessibilité
