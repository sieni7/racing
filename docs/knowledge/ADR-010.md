# ADR-010 : Conventions IA

**Statut** : Accepté  
**Date** : 2026-07-22  
**Approbation** : Équipe

## Contexte
Les agents IA (OpenCode / Cursor) doivent être guidés par des conventions explicites.

## Décision
- Rôles IA définis dans `.cursor/rules/agents.mdc`
- Chaque sprint liste les agents (Lead / Contributors / Validators)
- Les prompts incluent le contexte projet et les livrables attendus

## Conséquences
- Cohérence des interventions IA
- Documentation des décisions
- Facilitation des revues de code
