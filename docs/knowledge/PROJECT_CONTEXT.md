# Contexte projet

## Vision
Site officiel du Racing Club de Bingerville — club de football basé à Bingerville, Côte d'Ivoire.

## Stack
- **Frontend** : Vite 8 + React 18 + TypeScript 5 + Tailwind CSS 3
- **Backend** : Supabase (Auth, BDD, Storage)
- **Déploiement** : Netlify
- **Tests** : Vitest + Playwright

## Contraintes
- Connexion réseau parfois instable
- Budget limité (plan gratuit Supabase + Netlify)
- Projet en français (Côte d'Ivoire)

## KPIs
- Lighthouse Performance ≥ 90
- Lighthouse Accessibilité ≥ 90
- Temps de chargement < 2s
