# ADR-001 : Architecture MVC

**Statut** : Accepté  
**Date** : 2026-07-22  
**Approbation** : Équipe

## Contexte
Le projet doit avoir une architecture claire et maintenable.

## Décision
Séparer les responsabilités en couches :
- **Model** (`src/lib/`) : Services Supabase, logique métier
- **View** (`src/components/` + `src/pages/`) : Composants UI et pages
- **Controller** (`src/hooks/` + `src/contexts/`) : État global React, hooks personnalisés

## Conséquences
- Code organisé et réutilisable
- Facilite les tests (couches isolées)
- Data flow unidirectionnel
